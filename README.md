# Interactive RL Learning Web Tool

This is a web-based tool for exploring reinforcement learning algorithms and environments, built for educational purposes.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open http://localhost:5000 in your browser.

## Features
- Environments: StochasticGridWorld (custom), CartPole, MountainCar, FrozenLake, Breakout.
- Algorithms: Policy Iteration, Value Iteration, Monte Carlo, TD(0), n-step TD, SARSA, Q-Learning.
- Parameter adjustment via sliders.
- Visualizations: Value heatmaps, policy arrows, convergence plots, narrator explanations.
